# MacroZone Documentation

Welcome to the `macrozone` documentation hub.

This folder contains the primary documentation for the app, including the business logic, architecture overview, and feature descriptions.

## Documentation

- [Business Logic](business-logic.md) — explains how the meal logging, macro aggregation, reminders, and sharing features work.

## Project structure

- `src/app/` — Expo Router app routes and screens.
- `src/components/` — reusable UI components for macro cards, meal items, and controls.
- `src/storage/` — persistence layer for meals using AsyncStorage.
- `src/utils/` — notification scheduling and permissions helpers.

## How to use this docs folder

Use `README.md` at the repo root as the entrypoint, and follow links into `docs/` for deeper details.
