# Recurly Documentation

Welcome to the `recurrly` documentation hub.

This folder contains the primary documentation for the app, including the business logic, architecture overview, and feature descriptions.

## Documentation

- [Business Logic](business-logic.md) — explains how subscriptions, upcoming renewals, and home balance are modeled and presented.

## Project structure

- `app/` — Expo Router app routes and screens.
- `components/` — reusable UI components such as `SubscriptionCard` and `UpcomingSubscriptionCard`.
- `constants/` — data fixtures, icons, and helpers for formatting currency and dates.
- `app/(tabs)/index.tsx` — Home screen showing current balance, upcoming renewals, and active subscriptions.

## How to use this docs folder

Use `README.md` at the repository root as the entrypoint, and follow links into `docs/` for deeper details.
